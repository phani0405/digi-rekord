# Pre Requirements
1. node: v7.10.0(>=)
2. @angular/cli: v1.4.1

# Setup Instructions
1. Clone the project.
2. Go to project src.
2. RUN npm install
3. RUN npm start
4. Open http://localhost:4200

# Add MockData
1. npm install -g json-server
2. Go to projectFolder/src/api/institutes 
3. json-server --watch institutes.json
